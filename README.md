# Happy Birthday Sonakshi! 🎂🎈

A beautiful, interactive birthday web app dedicated to **Sonakshi**, built with HTML, CSS, and JavaScript.  
It features an improved UI, smart URL/search handling, and a playful experience designed to feel like a personal digital gift.

---

## Features

- Modern, mobile‑friendly UI with upgraded layout, colors, and animations.
- Dedicated birthday greeting page for **"Happy Birthday Sonakshi!"** with smooth transitions and effects.
- URL and search integration so you can open the birthday page directly from links or custom routes.
- Support for adding photos, messages, and surprises (confetti, balloons, etc.) depending on your implementation.
- Lightweight front‑end stack (pure HTML/CSS/JS) so it can run on any static host.

## Usage

- Open the main URL to see the "Happy Birthday Sonakshi!" homepage.
- Share the link with friends so they can view the surprise page.
- Customize text, colors, or images by editing `index.html`, `style.css`, and `script.js` (or the relevant files in your structure).

Example ideas:

- Change wishes in a `wishes` array in JavaScript.
- Swap background images or gradients in CSS.
- Hook different URLs (like `?mode=cute` or `/special`) to special variants of the page.
